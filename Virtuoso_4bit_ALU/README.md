# Cadence Virtuoso 활용 4bit ALU 설계 (4bit ALU Design using Cadence Virtuoso)

**기간**: 2025.06.03 ~ 2025.06.19

## 프로젝트 개요
교내 전공 과목을 수강하며 Cadence Virtuoso Tool을 활용하여 회로 설계부터 레이아웃까지 진행한 프로젝트입니다. 4bit ALU를 타겟으로 설계했습니다.

## 주요 내용
- **회로 설계**: CMOS Inverter부터 시작해 기본 게이트, Adder, Mux를 설계하고 이를 조합하여 ALU 구현
- **프로세스 경험**: 기초적인 아날로그 및 디지털 회로 설계 프로세스 경험

## 배운점
- 아날로그 회로 기반 레이아웃 계획 및 구현 역량 습득
- Cadence Virtuoso Tool 활용 능력 향상
