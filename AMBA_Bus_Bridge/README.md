# AMBA Protocol 기반 AHB-APB Bridge 설계 (AHB-APB Bridge Design based on AMBA Protocol)

**기간**: 2025.06.24 ~ 2025.07.14

## 프로젝트 개요
AMBA 프로토콜에 대한 이론 학습 후, 데이터시트를 기반으로 AHB-APB Bridge를 설계했습니다.

## 주요 내용
- **FSM 설계**: 서로 다른 데이터 전송 사이클을 맞추기 위한 1-wait FSM 설계
- **프로토콜 이해**: 통신 프로토콜에 대한 기초를 다짐
- **아쉬운 점**: AXI4 프로토콜에 대한 학습 부족

## 배운점
- AHB, APB 프로토콜에 대한 이해
- Verilog 코드 작성 기법 숙지
