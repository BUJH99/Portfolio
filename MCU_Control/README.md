# 레지스터 기반 MCU 제어 (Register-based MCU Control)

**기간**: 2024.09.01 ~ 2024.11.22

## 프로젝트 개요
레지스터 기반 MCU 제어 공모전 참가 프로젝트입니다. 아두이노 라이브러리 없이 데이터시트를 직접 분석하여 센서, MCU, 모터 등 다양한 부품을 제어했습니다.

## 주요 내용
- **Low-Level 제어**: 부품별 제어 코드를 라이브러리 없이 직접 구현했습니다.
- **시스템 통합**: 인터럽트와 폴링 방식에 대한 이해를 바탕으로 통합 시스템을 구현했습니다.
- **역공학 및 재설계**: 본선에서는 시중 전자제품을 오실로스코프로 분석하여 역공학을 진행하고, 의도대로 재설계했습니다.

### 담당 역할
- 모터 제어
- 인터럽트 구현
- LED 시각화 구현
- 전자제품 역공학 및 데이터시트 작성

### 직면한 어려움
- 폴링과 인터럽트 방식의 차이 및 적용
- Clock 제어 관련 기법
- 전력 및 물리적 핀 개수 부족 문제

## 배운점
- 폴링과 인터럽트 동작 원리 이해
- 데이터시트 활용 능력 및 기초적인 임베디드 코딩 플로우 습득
- 기초 부품 이해 및 활용 능력 향상
