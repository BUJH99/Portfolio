# 터치식 리듬게임 (Touch Rhythm Game)

**기간**: 2024.04.19 ~ 2024.05.05

## 프로젝트 개요

과학회 동아리 ’로보틱스’에서 진행한 프로젝트로, 파이썬 라이브러리인 Pygame과 아두이노를 활용하여 터치식 리듬게임을 제작했습니다.

## 주요 내용

- **시스템 설계**: 파이게임(Pygame)으로 리듬게임 로직을 구현하고, 아두이노와 금속판을 활용하여 임계전압(Vth) 인가 시 터치를 인식하도록 하는 통합 시스템을 설계했습니다.
- **기술 스택**: Python (Pygame), Arduino C, Hardware Interfacing

## 배운점

- 파이썬 프로그래밍 역량 향상
- 아두이노 활용 및 하드웨어 제어 역량 향상
