# 🎮 터치식 리듬게임 (Touch Rhythm Game)

## 📅 프로젝트 정보
- **진행 기간**: 2024.04.19 ~ 2024.05.05
- **기술 스택**: `Python (Pygame)`, `Arduino (C++)`, `Serial Communication`

---

## 📝 프로젝트 개요
아두이노 기반의 **정전식 터치 센서** 입력과 파이썬(**Pygame**) 기반의 게임 로직을 연동하여 제작한 터치식 리듬게임 프로젝트입니다.

## 🔑 주요 구현 내용
### 1. 하드웨어 및 인터페이스
- **터치 센서**: 금속판과 저항을 이용한 RC 회로 구성 및 전압 변화 감지 로직 구현.
- **통신**: Arduino와 PC 간 Serial 통신을 통해 실시간 입력 데이터 전송.

### 2. 소프트웨어 (Game Logic)
- **Pygame**: 노트 생성, 판정, 음악 재생, 점수 집계 등 게임 엔진 구현.
- **신호 처리**: ADC 데이터의 노이즈 필터링 및 터치 임계값(Threshold) 설정.

---

## 🚀 문제 해결 (Troubleshooting)
### 1. 터치 감도 불안정
- **문제**: 주변 환경(습도, 접촉 상태)에 따라 센서 값이 튀거나 오작동.
- **해결**: 실험을 통해 최적의 임계전압(Vth)을 설정하고, 소프트웨어적으로 데드존(Dead-zone)을 두어 노이즈 필터링.

### 2. 통신 지연 (Latency)
- **문제**: Serial 통신 속도 문제로 게임 판정 타이밍이 밀리는 현상.
- **해결**: Baudrate를 115200으로 상향하고 전송 데이터 패킷 크기를 최소화하여 지연 시간 단축.

---

## 📚 배운점
- **시스템 통합**: 하드웨어(센서)와 소프트웨어(게임) 간의 인터페이스 설계 및 데이터 연동 경험.
- **Serial 통신**: MCU와 PC 간 통신 프로토콜 설계 및 데이터 처리 방법 습득.
- **디버깅**: HW 신호 특성과 SW 로직을 동시에 분석하며 문제 해결 능력 배양.
