# C기반 AGV 활용 무인 주차장 시스템 시뮬레이터 (AGV Unmanned Parking System Simulator)

**수상**: 형남과학상 동상 수상
**기간**: 2025.03.01 ~ 2025.12.05

## 프로젝트 개요
C언어를 기반으로 AGV(무인 운반차)를 활용한 무인 주차장 시스템 시뮬레이터를 구현했습니다. C++에 대한 지식이 부족하여 C로 구현을 도전했습니다.

## 직면한 어려움
- **경로 탐색**: 관련 분야에 대한 이해 부족
- **충돌 회피**: 다개체 도입으로 인한 충돌 회피 시스템의 필요성
- **알고리즘 효율성**: MAPF(Multi-Agent Path Finding) 이해 부족과 단순 예외처리 반복으로 인한 비효율적인 연산 및 시스템 플로우
- **시스템 고도화**: 물리적 제약 도입으로 인한 교착 상태 발생 및 시스템 고도화 필요성
- **이식 실패**: 실제 로봇 이식 실패 및 중앙 제어 통신 미구현

## 성과 및 배운점
- 경로 탐색 알고리즘에 대한 기초 이론 및 구현 지식 습득
- 경로 탐색 알고리즘 성능 지표 설계 및 측정 능력 향상
- 기본적인 C언어 알고리즘 설계 능력 향상
- 성능 평가 공부를 통한 CS 지식 상승
