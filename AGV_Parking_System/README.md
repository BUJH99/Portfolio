# 🚙 C기반 AGV 무인 주차장 시스템 (AGV Parking System)

## 📅 프로젝트 정보
- **진행 기간**: 2025.03.01 ~ 2025.12.05
- **수 상**: 🏆 **형남과학상 동상**
- **기술 스택**: `C Language`, `WinAPI`, `Pathfinding Algorithms`

---

## 📝 프로젝트 개요
C언어와 WinAPI를 사용하여 다수의 **AGV(Automated Guided Vehicle)** 가 충돌 없이 최적 경로로 이동하는 무인 주차 시스템을 시뮬레이션으로 구현했습니다.

## 🔑 주요 레포트 내용 (알고리즘 구현)
### 1. 경로 탐색 알고리즘 (Pathfinding)
- **A* Algorithm**: 휴리스틱을 이용한 최단 경로 탐색 구현.
- **D* Lite**: 동적 장애물 발생 시 경로를 실시간으로 재탐색하는 알고리즘 적용.

### 2. 다개체 충돌 회피 (Multi-Agent)
- **WHCA* (Windowed Hierarchical Cooperative A*)**: 시간축(t)을 포함한 3차원(x, y, t) 공간 탐색으로 AGV 간 충돌 예측 및 회피.
- **CBS (Conflict Based Search)**: 개별 경로 탐색 후 충돌 발생 시 제약 조건을 추가하여 경로를 수정하는 방식 연구.

## 🚀 문제 해결 (Troubleshooting)
### 1. 교착 상태(Deadlock) 해결
- **문제**: 좁은 통로에서 여러 AGV가 마주칠 때 서로 비켜주지 못하고 멈추는 데드락 현상 발생.
- **해결**: 예약 테이블(Reservation Table)을 도입하여 특정 노드의 점유 시간을 미리 예약하도록 하여 충돌을 원천 차단.

### 2. 연산 속도 최적화
- **문제**: 다수의 AGV 경로를 매 프레임 계산 시 시뮬레이션 속도 저하.
- **해결**: 전체 경로를 매번 다시 계산하지 않고, 변경된 부분만 갱신하거나 일정 주기로만 재탐색하도록 최적화.
