# Softmax ACC IP 설계 및 SoC 구현 (Softmax Accelerator IP Design & SoC Implementation)

**수상**: 차세대반도체학과 주관 경진대회 우수상
**기간**: 2025.08.01 ~ 2025.09.25

## 프로젝트 개요
Softmax 연산 가속 IP를 설계하고 시스템에 통합했습니다.

## 주요 내용
- **IP 설계 및 통합**: RTL Coding뿐만 아니라 IP Integrator를 활용하여 SoC 구현
- **성능 향상**: 개발한 HW 기반 시스템 테스트 결과, C언어 기반 SW 대비 최대 13배 처리 속도 향상

### 담당 역할
- 일부 연산 모듈 설계
- SoC 구현

### 직면한 어려움 및 해결 노력
- SoC 구현 시 Tool 기반 IP에 대한 이해 부족
- Time Violation 및 CDC 문제
- SW 검증 방식 및 인터럽트 구현의 어려움
- 병목 지점 분석 및 PPA 최적화 코딩 기법 연구
